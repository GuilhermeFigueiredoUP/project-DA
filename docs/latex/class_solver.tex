\doxysubsection{Detailed Description}
Class to represent the solver for the assignment problem, which manages the graph representation of submissions and reviewers, parameters, and computation of assignments. 


\begin{DoxyParams}{Parameters}

\hline
{\em reviewers} & A vector of \doxylink{struct_data_node}{Data\+Node} objects representing the reviewers \\
\hline
{\em parameter\+Flags} & A bitmask representing the parameters for matching criteria \\
